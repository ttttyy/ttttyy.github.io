Matlab在高数线代学习中的应用(一)

本文最后更新于：2020年9月4日 晚上

前言

      Matlab是一个功能十分强大的数学软件，如果在高数和线代学习中可以熟练使用Matlab解决一些繁杂的计算问题，验证手工计算的正确性，对学习大有裨益。但是网络上的大多数博文都只是对某个问题提出解决方案，并没有一个完整的总结，给搜索带来了不便，因此，本系列博文将对高数和线代中Matlab的应用做一总结，方便读者查阅。因本人水平所限，博文中存在一些不当之处，欢迎各位读者在评论区提出批评和指正。

高数

极限

  1. 计算$\displaystyle\lim_{x \rightarrow + \infty}(1+\frac{1}{x})^x$

    syms x
limit((1+1/x)^x,x,inf)   
%第一个参数为函数表达式，第二个参数为自变量，第三个参数为自变量趋向的值
%结果
ans=exp(1)

  2. 计算$\displaystyle\lim_{x \rightarrow 3^+}f(x)=\frac{x-3}{|x-3|}$的极限

    syms x
limit((x-3)/abs(x-3),x,3,'right')
%right指示右极限,left指示左极限
ans=1

微分

  1. 求$f(x)=arcsin(\frac{2x}{1+x^2})$的导数

    syms x f(x)
f(x)=asin(2*x/(1+x^2));
ans=diff(f,x,1)
pretty(ans)
ans=
2        4 x
------ - ---------
2         2     2
x  + 1   (x  + 1)

    /           2   \
    |        4 x    |
sqrt| 1 - --------- |
    |       2     2 |
    \     (x  + 1)  /
%可以用pretty()函数对结果进行美化

  2. 求$f(x)=e^x+1$的反函数

    syms x f(x)
f=exp(x)+1;
finverse(f,x)
ans = log(x-1)

  3. 求$\frac{\partial^2 u}{\partial x \partial y},u=f(x,y)=xsin(xy)$

    syms x y
diff(x*sin(x*y), x, y)
ans=2*x*cos(x*y) - x^2*y*sin(x*y)

  4. 设$siny+e^x-xy^2=0$,求$\frac{\mathrm{d} y }{\mathrm{d} x}$

    syms x y f
f=sin(y)+exp(x)-x*y^2;
ans=-diff(f,x)/diff(f,y);
pretty(ans)
ans=
    2
- y  + exp(x)
- --------------
cos(y) - 2 x y

  5. 求$f(x)=ln(x)$按$(x-1)$的幂展开的6阶泰勒公式

    syms x f(x)
f=log(x);
ans=taylor(f,x,1,'order',7); %忽略7阶及以上小量
pretty(ans)

        2          3          4          5          6

积分

  1. 求$f(x)=\int{xcos^2x}dx$

    syms x f
f=x*(cos(x))^2;
ans=int(f,x);
pretty(ans)
ans=
                2    2
x sin(2 x)   sin(x)    x
---------- - ------- + --
    4          4       4

  2. 计算$\int_{0}^{\frac{\pi}{2}}{\frac{x+sinx}{1+cosx}}dx$

    syms x f
f=(x+sin(x))/(1+cos(x));
ans=int(f,x,0,pi/2);
ans=pi/2

微分方程

Dy代表y的导数，D2y代表y的二阶导数， D3y代表y的三阶导数…

  1. 求解微分方程$\frac{\mathrm d y}{\mathrm d x}=10^{x+y}$

    y=dsolve('Dy-10^(x+y)=0','x');
pretty(y)
y=
            x
log(C1 - 10 )
- -------------
    log(10)

  2. 求微分方程$(1+x^2)y^{\prime\prime}=2xy^{\prime}$满足初值条件$y|_{x=0}=1,y^{\prime}|_{x=0}=1$的特解

    y=dsolve('(1+x^2)*D2y-2*x*Dy=0','y(0)=1,Dy(0)=3','x');
pretty(y)
y=
    2
x (x  + 3) + 1

画函数图像

  1. 画箕(ji)舌线$y=\frac{8}{x^2+4}$的函数图像 
    x=linspace(-6,6,100);
y=8./(x.^2+4);
plot(x,y)

  1. 画出$x^2+y^2+2x=2\sqrt{x^2+y^2}$的函数图像

    fimplicit(@(x,y) x.^2+y.^2+2*x-2*(x.^2+y.^2)^0.5)

  1. 画出$\rho=2sin2\theta$的图像

    theta = 0:0.01:2*pi;
rho=2*sin(2*theta);
polar(theta,rho)

  1. 画出双曲抛物面$\frac{x^2}{2^2}-y^2=z$的图像

    f = @(x,y,z) x.^2/4 - y.^2 - z;
fimplicit3(f,'EdgeColor','none','FaceAlpha',.5)
